## Reglas

- **Hechos verificables**, no marketing inflado.
- Si falta info, marcar `[FALTA: <pregunta>]` — el `brand-researcher` lo completa o pregunta.
- Citar fuentes externas cuando aplique (URL, fecha).
- Mantener cada archivo entre 500-2000 palabras. Más allá → dividir en sub-secciones.
